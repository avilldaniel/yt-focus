# YT Focus

Chrome extension that auto-focuses the YouTube video player when the browser window regains focus, enabling immediate keyboard control (spacebar, arrow keys, etc.) without clicking.

Works on youtube.com and music.youtube.com.

## Install

```sh
pnpm install
pnpm dev
```

Load the extension from `.output/chrome-mv3` in `chrome://extensions` with developer mode enabled.

## Build

```sh
pnpm build
```
